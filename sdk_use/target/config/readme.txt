//////////////////////////////////////////////////////////////////////////////
目录说明
//////////////////////////////////////////////////////////////////////////////
hsocfg      : HSO插件的显示配置文件(XML文件)
app         : 应用层的HSO配置数据的接口定义文件
hdbcfg      : HSO 配置数据的显示配置文件(XML文件)
diag        : 模块的HSO配置数据的接口定义文件(HSO包定义)
hso         : 模块的HSO配置数据的接口定义文件(基本类型定义)
nv          : NV配置(包括HSO的显示配置和接口定义文件)
product     : 产品发布的配置文件,包括产品类型,单板类型,芯片类型,软件特性开关的定义
stub        : 开发过程中相关桩接口定义
