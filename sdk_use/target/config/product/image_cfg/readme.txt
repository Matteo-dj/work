****************************************************************************
* 镜像文件生成配置
* 包括 .hbin .hupg文件生成、发布包生成的内容配置
* 前缀不能包含 . 字符
* .cfg文件名格式为    <前缀>.hupg.<产品版本>.cfg  产品版本即为显示在OneBuilder的Product下的菜单项名称，注意必须为小写， 如 himdl_v100r001
* .packet文件名格式为 <发布包名称>.xxx.packet
****************************************************************************

<前缀>.hupg.himdl_v100r001.cfg  : 表示需要生成 himdl_v100r001 产品版本的 .hupg、.hbin文件, 包含的内容在该文件中配置

<前缀>.hupg.hbin.packet         : 表示发布包<前缀>中包含的.hupg和.hbin文件配置

<前缀>.dbg.packet                : 表示发布包<前缀>中包含的调试使用的目录配置

****************************************************************************

****************************************************************************
注意: 
通过custom build 菜单 选择MRS 镜像后，仅支持 config_firmware_mrs_image.conf文件中的.hbin .hupg
通过custom build 菜单 选择IPMRS 镜像后，仅支持 config_firmware_ipmrs_image.conf文件中的.hbin .hupg
通过custom build 菜单 选择EQT 镜像后，仅支持 config_firmware_eqt_image.conf文件中的.hbin .hupg
****************************************************************************
