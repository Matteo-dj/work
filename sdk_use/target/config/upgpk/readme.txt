****************************************************************************
* 自定义升级文件的配置目录
****************************************************************************
规则
1、目录名称表示产品名称，即OneBuilder的Product菜单下的菜单项名称(小写)
2、目录内存放upgpk的配置文件，参见himdl例子
3、对应该目录名下的产品，会自动生成SDK包中APP的upgpk
     例如himdl有mrs、ipmrsAPP则 SDK中的upgpk会自动生成 mrs、ipmrs目录
